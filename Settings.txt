JavaFX설정
Properties(Alt+Enter) -> Run/Debug Setting -> Edit...
Arguments->VM arguments: (아래문장, 경로에 유의하세요.)

--module-path "\javafx-sdk-16" --add-modules=javafx.controls,javafx.fxml

* User Library추가
Window -> Preferences ...
java -> Build Path -> User Libraries ...
New... "JavaFX"
"JavaFX"선택 -> Add External JARS...
(자신이 설치한 폴더)\javafx-sdk-16\lib...
.jar만 선택(복수선택가능, .zip은 선택하면 안됨)

Build -> Configure Build path...
libraries -> Add labrary -> User Library -> JavaFX선택 -> Finish
